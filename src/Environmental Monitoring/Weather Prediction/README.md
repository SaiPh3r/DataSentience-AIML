# Weather Prediction

## ARIMA : 
The ARIMA model is used to forecast the weather. 
Accuracy of the model was calculated using the root mean square error (RMSE). 
It is observed that the RMSE estimated for forecasting using ARIMA was 2.24.

## Libraries required :
- Numpy 
- Pandas
- Matplotlib 
- Sklearn    
- Seaborn 
- Statsmodels
- Pmdarima
